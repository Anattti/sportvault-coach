Live Activityn ja Dynamic Islandin ulkoasua (designia) pääset muokkaamaan tiedostosta:
👉 **`src/widgets/WorkoutLiveActivity.tsx`**

Tämä tiedosto käyttää SwiftUI:n elementtejä (kuten `HStack`, `VStack`, `Text`, `Spacer`) suoraan JavaScript-puolelta `expo-widgets` -kirjaston avulla.

Tiedosto on jaettu seuraaviin palasiin, joita muokkaamalla saat muutettua eri näkymiä:

### 1. Lukitusnäyttö (Lock Screen Banner)
Kohdasta `banner:` löytyy se laatikko, joka näkyy puhelimen lukitusnäytöllä, kun treeni on käynnissä. Täällä voit muuttaa esimerkiksi taustaväriä, tekstien fontteja ja asettelua (nyt se näyttää mm. liikkeen nimen ja sarjojen määrän).

### 2. Dynamic Island (Laajennettu näkymä)
Kohdasta `expanded:` löytyvät asetukset sille, kun käyttäjä painaa Dynamic Islandia pitkään ja se laajenee isoksi kortiksi näytön yläreunaan. Se on jaettu neljään alueeseen:
- `leading`: Yläkulman vasen puoli
- `trailing`: Yläkulman oikea puoli
- `center`: Keskikohta
- `bottom`: Alaosan leveä alue (nyt siellä näkyy syke ja kulunut aika)

### 3. Dynamic Island (Pienennetty / Compact näkymä)
Kohdasta `compactLeading:` ja `compactTrailing:` löytyvät ne kaksi pientä palaa, jotka näkyvät Dynamic Islandin sivuilla, kun sovellus on taustalla ja saareke on normaalissa kapeassa tilassaan. Nyt vasemmalla (`compactLeading`) näkyy liikkeen nimi ja oikealla (`compactTrailing`) kulunut aika.

### 4. Dynamic Island (Minimaalinen näkymä)
Kohdasta `minimal:` löytyy kaikista pienin näkymä, joka on näkyvissä silloin, jos taustalla on myös toinen Live Activity (esim. ajastin tai Uber-kyyti).

### Miten muokkaat (Tyylit)
Tyylien tekeminen poikkeaa hieman React Nativesta. Voit tuoda kaikki tarvittavat tyylit `modifiers`-paketista, esimerkiksi:
```tsx
import { font, foregroundStyle, padding, background, cornerRadius } from '@expo/ui/swift-ui/modifiers';
```
Ja liittää niitä elementteihin `modifiers`-taulukon avulla:
```tsx
<Text modifiers={[foregroundStyle("white"), font({ weight: "bold" })]}>
  Treeni
</Text>
```

Voit vapaasti testata uusia asetteluita tuossa tiedostossa. Aina kun tallennat, Metro-bundler lataa muutokset (joskus Live Activity vaatii täyden uudelleenkäynnistyksen, jotta monimutkaisimmat UI-muutokset päivittyvät saarekkeeseen).
